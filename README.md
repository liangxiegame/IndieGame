# IndieGame



## 做游戏的时候

切换思维

* 技术思维
* 设计角度（策划思维）
* 美术/音乐角度

## 一个游戏的结构

* 关卡有结束
* 惩罚
  * 角色死亡
  * 血量
  * 重玩
  * 生命数量减掉 1 
* 成就感





## 关卡做的事情

| 关卡   | 职责                          |

## 技巧

* 按下之后不松手，到达跳跃的最高点之后再马上按第二次跳跃如此往复。



## 玩家跳跃能力的等级：

* 跳跃,单方向。
* 能够达到二段跳的高度，加上左右移动。
* 三段跳的高度，最高高度。
